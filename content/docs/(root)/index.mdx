**Sidekick** is a collection of AI-focused UI components designed for building chat interfaces and conversational experiences.

## What's Included

- **Sidekick**: A collapsible AI chat sidebar with conversation history and message bubbles
- **PromptInput**: An advanced input component with file attachments and speech support

## Installation

Pick the components you need and add them to your project using the CLI:

```bash
npx shadcn@latest add https://sidekick.dev/r/sidekick.json
```

```bash
npx shadcn@latest add https://sidekick.dev/r/prompt-input.json
```

Or copy and paste the code directly into your project.

## Open Source

All components are open source. The code is yours to customize and extend to fit your needs.
