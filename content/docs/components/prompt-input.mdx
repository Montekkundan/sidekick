## Usage

```tsx
import {
  PromptInput,
  PromptInputBody,
  PromptInputTextarea,
  PromptInputFooter,
  PromptInputTools,
  PromptInputSubmit,
} from "@/components/prompt-input";
```

```tsx
<PromptInput onSubmit={(message) => console.log(message)}>
  <PromptInputBody>
    <PromptInputTextarea />
  </PromptInputBody>
  <PromptInputFooter>
    <PromptInputTools />
    <PromptInputSubmit />
  </PromptInputFooter>
</PromptInput>
```

## Features

- **File Attachments**: Drag and drop or paste files directly into the input
- **Speech Recognition**: Built-in speech-to-text support
- **Model Selection**: Dropdown for selecting AI models
- **Status Indicators**: Shows loading, streaming, and error states
- **Keyboard Shortcuts**: Submit with Enter, Shift+Enter for new line

## API Reference

### PromptInput

The root component that handles form submission and file management.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `onSubmit` | `(message: PromptInputMessage) => void` | required | Called when the form is submitted |
| `accept` | `string` | - | Accepted file types (e.g., "image/*") |
| `multiple` | `boolean` | `false` | Allow multiple file uploads |
| `globalDrop` | `boolean` | `false` | Accept drops anywhere on the document |
| `maxFiles` | `number` | - | Maximum number of files allowed |
| `maxFileSize` | `number` | - | Maximum file size in bytes |

### PromptInputSubmit

The submit button with status indicator.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `status` | `"ready" \| "submitted" \| "streaming" \| "error"` | `"ready"` | Current status of the submission |
